# Rewards and Promotion Codes

## How to Register a Promo Code
If you have a promotion code (e.g., for a free transfer), follow these steps to register it:

1. Open the **Brastel Remit** app.
2. Tap **Menu** (3 horizontal bars).
3. Select **Rewards**.
4. Enter the promo code and tap **SUBMIT**.
5. Tap **OK** when the “Registration Successful” message is displayed.

## Important Rules
- **Timing**: The promo code **must be entered before your first money transfer**.
- **Validity**: Promo codes are valid for **6 months** from the date of registration.
- **Benefits**: Usually provides a discount or a completely free transfer (waiving the Brastel Remit transfer fee).

## Inviting Friends
1. Tap **Menu** -> **Rewards**.
2. Tap on your promo code.
3. Select an app to share your unique code with your contacts.
4. When your friend signs up and uses your code for their first transfer, you may receive rewards.
